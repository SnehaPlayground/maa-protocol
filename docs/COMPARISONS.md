# Comparisons

## Maa Protocol vs LangGraph
LangGraph handles workflow construction and execution. Maa Protocol wraps those workflows with approval, cost, tenant, resilience, and observability controls.

## Maa Protocol vs full agent frameworks
Frameworks such as CrewAI or AutoGen focus on agent behavior and orchestration. Maa Protocol is intentionally narrower. It is about control, not agent personality or planning strategy.

## Maa Protocol vs internal platform code
If your team already has homegrown budget checks, approval ledgers, and tenant isolation layers, Maa Protocol aims to package those concerns cleanly rather than forcing a new runtime.
