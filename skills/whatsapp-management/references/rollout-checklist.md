# Rollout Checklist

## Phase 1: Build
- create framework files
- create business knowledge pack
- define owner commands
- define logging schema

## Phase 2: Validate locally
- confirm runtime target agent
- confirm WhatsApp binding
- confirm Telegram owner identity
- confirm logs append safely

## Phase 3: Functional tests
- DM hello test
- DM qualification test
- group silent test
- group tagged in-scope test
- group tagged out-of-scope test
- owner command test
- failure alert test

## Phase 4: Soft launch
- monitor first real conversations
- tune playbooks and KB
- review false positives / false replies

## Phase 5: Template readiness
- remove business-specific assumptions from engine
- keep business pack portable
- document onboarding inputs needed for another business
